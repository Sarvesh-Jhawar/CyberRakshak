# 🧠 LLM Orchestration Layer

This directory contains the logic for interacting with Large Language Models (LLMs) used in the CyberRakshak project.

## 🚀 Overview

The LLM layer is responsible for:
- **Threat Intent Routing**: Categorizing user input to trigger the correct Machine Learning models.
- **Incident Synthesis**: Merging probabilistic ML scores with heuristic reasoning to generate comprehensive reports.
- **Natural Language Understanding**: Processing user descriptions of cyber incidents.

## 📁 Key Files

- `main2.py`: Contains experimental or primary orchestration logic for LLM processing.
- `image.png`: Architecture visualization or design asset.

## 🔌 Integration

This layer typically connects to APIs like **Groq** (using Llama 3 or Mixtral models) to provide high-speed, intelligent threat analysis.
