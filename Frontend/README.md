# 🌐 CyberRakshak: Frontend Interface

This is the frontend component of the CyberRakshak project, built with modern web technologies to provide a premium, high-performance user experience.

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Components**: Shadcn UI & Radix UI
- **Icons**: Lucide React
- **Animations**: Tailwind-animate
- **Charts**: Recharts

## 📁 Directory Structure

- `app/`: Contains the main pages and routing logic.
- `components/`: Modular UI components (Dashboard, Chat, etc.).
- `hooks/`: Custom React hooks for state and API calling.
- `lib/`: Utility functions and shared libraries.
- `public/`: Static assets and images.
- `styles/`: Global CSS and styling configurations.

## 🚀 Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Run Development Server**:
    ```bash
    npm run dev
    ```
3.  **Build for Production**:
    ```bash
    npm run build
    ```

## 🔌 API Integration

The frontend connects to the Backend API (FastAPI) typically running on `http://localhost:8000`. Ensure the backend is active before using the dashboard features.
